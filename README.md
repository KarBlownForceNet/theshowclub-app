# MLB Companion Repository

This repository houses downloadable files for the MLB Companion project, including the Windows installer, Chrome extension, and other release assets.

## Downloads

All downloadable files are available in the [Releases](https://github.com/KarBlownForceNet/theshowclub-app/releases) section of this repository. Each release contains:

- Windows installer (`MLB-Companion-Setup-<version>.exe`)

- Chrome extension (`mlb-companion-extension.zip`)

- Version metadata (`latest.yml`)

- Any additional release assets

To download:

1. Go to the [Releases](https://github.comKarBlownForceNet/theshowclub-app/releases) page

2. Select the latest release (or the specific version you need)

3. Download the assets from the release page

## Usage

## Windows Installer

1. Download `MLB-Companion-Setup-<version>.exe` from the latest release

2. Run the installer and follow the on-screen instructions

3. The application will be installed and ready to use

## Chrome Extension

1. Download `mlb-companion-extension.zip` from the latest release

2. Extract the ZIP file to a folder on your computer

3. Open Chrome and go to `chrome://extensions`

4. Enable "Developer mode" in the top right

5. Click "Load unpacked" and select the extracted extension folder
